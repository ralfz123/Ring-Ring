## :rocket: Install project

#### Clone repository

```bash
git clone https://github/com/joordy/ringring.git
```

#### Install project

```bash
npm i
```

#### Run the project

```bash
python -m SimpleHTTPServer 8000  
```

#### Visit project

```bash
localhost:000
```
